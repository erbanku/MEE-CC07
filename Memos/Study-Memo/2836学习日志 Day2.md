Part 1
闪灯程序练习
编写了一个可以每点按一次开关，在长亮、闪亮、长灭间切换的程序并利用树莓派和面包板实现。

```python
import RPi.GPIO as GPIO
import time
GPIO.setmode(GPIO.BCM)
GPIO.setmode(GPIO.IN)
GPIO.setup(25,GPIO.OUT)
n=0
while True:
	inputValue=GPIO.input(24)
	if(inputValue===True):
		n=(n+1)%3
	if n==1:
		GPIO.output(25,GPIO.HIGH)
		time.sleep(1)
	elif n==2:
		GPIO.output(25,GPIO.HIGH)
		time.sleep(1)
		GPIO.output(25,GPIO.LOW)
		time.sleep(1)
	else:
		GPIO.output(25,GPIO.LOW)
		time.sleep(1)
```
